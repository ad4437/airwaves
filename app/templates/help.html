Help

Graphing Programs

All graphs use the Plotly Graphing Framework

Basic Usage:
There are three types of signal measurements monitored by a HDHomeRun tuner: signal strength (ss), signal quality (snq), and symbol (picture) quality (seq). HDHomeRun provides an overview of what these mean and how to use them tl;dr: Signal quality best describes a signal's clarity, signal strength is somewhat irrelevant, and picture quality is either 0 or 100, with 100 indicating a watchable signal and 0 a static signal.

These signal measurements are monitored and recorded for each real channel picked up by the antenna. Virtual channel numbers are mapped to each real channel, hence their grouping in channel labels. Further reading on real channels can be found here, and virtual channels can be found here. Channel labels are composed of real channels and all virtual channels and their stations that map to it.

Hovering over a data point will trigger its tooltip, displaying its x and y data values and its corresponding legend entry label.
Legends are displayed to the right of the graph. The Legend can be used to toggle the visibility of a data set. Clicking a legend entry once will hide its corresponding data set; clicking it a second time will return the data set to the graph.

The Modebar:
The modebar appears in the upper right corner of the graph when hovered over. The modebar contains buttons that alter the graph usage and view, but not the data.

In addition to plotly functionality, there are input fields underneath the graphs that modify the data displayed. These inputs vary in usage and function between graphing programs, but the one consistent input field is the Antenna Selectbox.

Antenna Selectbox:
By default, the selectbox is set to the configured antenna instance (currently monitored with hourly scans). Antenna instance labels are composed by their unique ID, name, location (within the house), direction (in degrees), and any other comments included in the antenna entry.


Track Channel

Plotting signal measurements of channels over time.

Signal Measurement Radio Buttons:
Signal quality is the default signal measurement, but this can be changed with the radio buttons. The possible signal measurements are signal quality (snq), signal strength (Ss), and picture quality (seq).

Other Buttons:
The "Legend Scrollview" button can be used to alternate between a scrollview and fullview of the legend. The "Legend Fullview" displays all channels at the same time. Clicking this button will alternate between the "Legend Scrollview" and "Legend Fullview" texts. The "Hide All" button can be used to hide all of the channel line traces at once. If all traces are hidden, the text will update to "Show All," and can be used to set all traces to visible.

Hovering over a data point will trigger a tooltip containing the timestamp, signal measurement value, and available weather data reported at the time of that measurement.


Channel Distribution

Plotting signal measurement distributions for various channels.

Channel Selectbox:
By default, the selectbox is set to the lowest real channel number. To switch between channels, you can use the selectbox to find the wanted channel, a search box is provided with the drop-up/down (Easier to find virtual channels / stations). For faster channel selection, use the key to select the next lowest real channel and the key to select the next highest real channel.

Frequency Distribution Radio Buttons:
The Frequency Distribution y-data is automatically displayed as scan counts (Frequency). To use a Relative Frequency Distribution, select "Relative Frequency". A Relative Frequency will use a percentage of the total number of scans as the y-data.

Model Type:
Models are provided as an estimation of a channel's signal measurement distributions. Two types of models are provided, Gaussian Kernel Density Estimation and Normal Distribution Models. Use this radio button to switch between the two models. Models for picture quality (seq) are either 100% at measurements of 0 or 100 or split between the two. For channels that are consistently recepted, the signal strength will typically be 100 most of the time. This creates very tall modes at 0 and 100 along the x-axis and often leads to not so useful models. However, signal quality measurements are typically varied, so the most useful models will fit a channels signal quality.

Hide Models:
This button will toggle model visibility. Click it once to hide all models, and click it a second time to make them reappear.

Filtering the Data:
The filter button will open a filter prompt window with available filter conditions. When there are no filter conditions applied to the data, the filter button will be set to unactive (gray button), but active (green button) when filter conditions have been applied.



Signal measurement distributions can be filtered by Time or Weather Conditions.


Filter Input Function
Time of Day Only select scans from each day that ran during the designated time period i.e. select scans from 6AM to 6PM vice versa
Scan Date Range Only select scans within the designated date range i.e. between 6/22/2020 and 7/21/2020
Temperature Range Only select scans that report a temperature (in Farenheit) within the designated temperature range i.e. between 80°F and 90°F
Wind Speed Range Only select scans that report a wind speed (in miles per hour) within the designated wind speed range i.e. between 10mph and 20mph
Wind Direction Range Only select scans that report a wind direction (in degrees) within the designated wind direction range i.e. between 150° and 180°
Humidity Range Only select scans that report a humidity percentage within the designated humidity range i.e. between 80% and 100% humidity
Weather Status Only select scans that report a weather status (forcast) matching the selected weather status


Interact with time or date pickers by clicking the clock or calendar icons; A date/time picker should appear. To apply filter conditions to a data set, use the "Apply" button. To remove filter conditions, re-enter the filter window and press the "Clear All Filters" Button. Filters applied to one channel/antenna instance will be applied to any other selected channels or antennas until the filters are cleared.


Scan Summary

Plotting channel signal measurements for an individual scan.

Scan Start Time:
This datetime picker will let you pick the date and time of any scan run for the selected antenna instance. The datetime picker will appear after clicking the calendar icon. When selecting a datetime, the scan taken the closest to the selected time will be graphed. Scans are automatically run 7 minutes past the hour, but manual scans are also available for selection. To quickly switch between scans, use the key to select the same time on the previous day and the key to select the same time on the next day while the datetime picker is open. To jump one week between scans, use the and keys.
